# Project Memory

## Core
LandingCraft AI: Editorial-grade landing page generator with taste-skill applied.
Strictly use Anime.js for ALL animations (app UI & generated pages).
Tech Stack: Supabase Edge Functions, Lovable AI Gateway (Gemini).
Persist history locally via localStorage; no backend DB.
Previews must use sandboxed iframe (srcDoc) for isolated execution.
Never use Lorem Ipsum, banned phrases (Elevate, Seamless, Unleash...), or generic AI patterns.
App UI: Instrument Serif + Geist, off-black/warm-cream, single warm-amber accent. Never pure #000/#FFF, never purple-blue gradient.
